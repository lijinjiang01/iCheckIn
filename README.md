## 介绍
iCheckIn 为微信小程序项目，通过调用手机蓝牙功能搜索 Beacon 设备，实现课程签到的功能，该项目我只完成了微信小程序端的实现，流程设计和原型设计都是参考我们小组组长给的样例，重新设计的。这里是他的[GitHub主页链接](https://github.com/tokgocode)，有兴趣大家可以去点下关注
